# Join Build For Freedom — Static Site

This folder contains your production-ready website for **joinbuildforfreedom.com**.

## Deploy to GitHub + Vercel (no code)
1. Create a new GitHub repo named `joinbuildforfreedom`.
2. Click **Add file → Upload files** and upload this entire folder (including `/assets/images`).
3. Go to **Vercel → Add New Project → Import Git Repository**, choose your repo.
4. In project settings: Framework **Other** · Build Command **(leave blank)** · Output Directory **.**
5. Click **Deploy**. Your site will be live at a `.vercel.app` URL.
6. In **Vercel → Settings → Domains**, add `joinbuildforfreedom.com` and follow DNS prompts:
   - A `@` → `76.76.21.21`
   - CNAME `www` → `cname.vercel-dns.com`

## Payment Links
Replace the `href="#"` values in the pricing buttons with your live Stripe/Stan checkout links.
